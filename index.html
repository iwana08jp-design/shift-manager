📖 セットアップ手順
1
LINE Developersでチャネル作成
LINE Developers で「Messaging API」チャネルを作成し Channel Secret を取得します。
2
長期アクセストークンを発行
「Messaging API設定」タブで長期チャネルアクセストークンを発行し、左の欄に入力します。
3
Webhookサーバーをデプロイ（Render等）
// npm install express @line/bot-sdk const line = require('@line/bot-sdk'); const express = require('express'); const config = { channelSecret:'SECRET', channelAccessToken:'TOKEN' }; const client = new line.Client(config); const app = express(); app.post('/webhook', line.middleware(config), (req,res) => { req.body.events.forEach(e => { if(e.type==='message') handleMsg(e); }); res.sendStatus(200); }); app.post('/notify', express.json(), async (req,res) => { await client.pushMessage(req.body.groupId,{type:'text',text:req.body.message}); res.json({ok:true}); }); app.listen(3000);
4
Webhook URLをLINE Developersに登録
デプロイしたURL（例: https://yourapp.onrender.com/webhook）を登録し「検証」します。
✓
完了！テスト送信で動作確認
設定保存後「テスト」ボタンで通知が届くか確認してください。
